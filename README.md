# CamExtParamEst
This is an external parameters estimation project for a camera in a scene dominated by approximate plane motion.  

#### Environment
Dependences: ceres-solver, Eigen  
It was only tested in Ubuntu 18.04.

#### Usage
**For build**
```
mkdir build  
cd build  
cmake ..  
make  
```

**For run**
```
cd bin  
./main  
```
